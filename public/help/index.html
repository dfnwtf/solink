üîí Security & Privacy

Is SOLink secure?

Yes. SOLink uses end-to-end encryption (E2E) with NaCl cryptography (XSalsa20-Poly1305). This means:

  • Messages are encrypted on your device before sending
  • Only you and the recipient can read messages
  • We cannot see your messages ‚Äî even if we wanted to
  • No backdoors, no exceptions
Do you store my private keys?

No, never. Your private keys stay in your wallet (Phantom, Solflare, etc.) and never leave your device. We only see your public wallet address.

Where are my messages stored?
  • Locally: Your message history is stored in your browser (IndexedDB)
  • Temporarily on server: Encrypted messages wait on our server until delivered, then are deleted

We only store encrypted ciphertext ‚Äî we can't read your messages.

How do I backup my messages?

Go to Settings ‚Üí Backup & Restore:

  • Create Backup: Exports your messages as an encrypted file (AES-256)
  • Restore Backup: Import your messages on a new device

Set a strong password ‚Äî without it, the backup cannot be restored.

What is the Token Scanner?

The Token Scanner analyzes Solana tokens and provides security risk assessments powered by DFN Patrol. It checks:

  • Contract security (mint authority, freeze authority)
  • Liquidity status and top holders
  • Social presence and community
  • DEX payment status
  • Potential rug pull indicators
How do I scan a token?

Two ways:

  • Scan tab: Click the üîç icon, paste a token address or link
  • In chat: Paste any token link (pump.fun, dexscreener, etc.) ‚Äî it auto-generates a preview card
Should I trust the risk score?

Use it as one data point, not the only source.

The scanner provides algorithmic analysis based on available data. It may not detect all risks. Always:

  • Do your own research (DYOR)
  • Check multiple sources
  • Never invest more than you can afford to lose

üîß Troubleshooting

Wallet won't connect

Try these steps:

  • Make sure your wallet extension is installed and unlocked
  • Refresh the page and try again
  • Check if you have multiple wallet extensions ‚Äî disable conflicting ones
  • Try a different browser (Chrome works best)
  • Clear browser cache and cookies for solink.chat
Messages not sending

If messages show "Sending..." but never deliver:

  • Check your internet connection
  • Refresh the page
  • Make sure you're still connected to your wallet
  • Try disconnecting and reconnecting your wallet
Push notifications not working

Check these settings:

  • Make sure notifications are enabled in SOLink Settings
  • Check browser notification permissions for solink.chat
  • On mobile: check system notification settings
  • Notifications only work when the app is closed or in background
Lost my chat history

Chat history is stored locally in your browser. If you:

  • Cleared browser data: History is lost unless you have a backup
  • Using a new device: Use Restore Backup if you have one
  • Using incognito/private mode: Data is deleted when you close the window

Tip: Create regular backups in Settings ‚Üí Backup & Restore
